smss.exe - Session Manager Subsystem

Location: services/smss/smss.c
Type: C Executable
Purpose: Starts user session and loads critical system processes

Overview

smss.exe (Session Manager Subsystem) is responsible for starting the user session, loading critical system processes, and handling further stages of the boot process.

Main Function

int main(int argc, char** argv) {
    printf("FreeWorld Session Manager Subsystem (SMSS) v0.1\n");
    printf("Starting user session...\n");
    
    // Initialize session
    // Load critical system processes
    // Start CSRSS
    // Start login manager
    
    printf("Session started successfully.\n");
    
    // Start login process
    system("freeworldlogon.exe");
    
    return 0;
}

Responsibilities

  • Initialize user session
  • Load critical system processes
  • Start CSRSS (Client Server Runtime Subsystem)
  • Start login manager (freeworldlogon.exe)

Boot Sequence

  1. Kernel starts SMSS
  2. SMSS initializes session
  3. SMSS loads critical processes
  4. SMSS starts CSRSS
  5. SMSS starts login manager

Integration Points

  • fwoskrnl.exe: Started by kernel after initialization
  • csrss.exe: Started by SMSS
  • freeworldlogon.exe: Started by SMSS
