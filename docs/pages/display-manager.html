Display Manager

Graphical login and session management

Navigation

  • Home
  • Display Manager

Overview

The Display Manager provides the graphical login screen and manages user sessions, starting the desktop environment after authentication.

Functions

  • Display login screen
  • User authentication
  • Session management
  • Desktop environment startup
  • Session switching

Integration

The Display Manager integrates with FreeWorld Logon for authentication and starts the desktop environment (Wanderer) after successful login.
