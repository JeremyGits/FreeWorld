Boot Process

Purpose: Complete boot sequence documentation

Overview

The FreeWorld OS boot process follows a multi-stage sequence from BIOS/UEFI initialization to user login.

Boot Sequence

  1. BIOS/UEFI Initialization
    • BIOS/UEFI performs hardware initialization
    • BIOS/UEFI loads boot sector from disk
  2. BOOTMGR (Boot Manager)
    • BIOS loads boot sector (512 bytes) to 0x7C00
    • BOOTMGR initializes segments and stack
    • BOOTMGR displays boot message
    • BOOTMGR loads kernel (via load_kernel)
    • BOOTMGR jumps to kernel entry point (0x1000:0x0000)
  3. BCD (Boot Configuration Data)
    • freeload.exe reads BCD file
    • BCD provides boot entry configuration
    • Contains kernel and loader paths
  4. freeload.exe (OS Loader)
    • Loads BCD configuration
    • Loads kernel (fwoskrnl.exe) into memory
    • Loads essential device drivers
    • Transfers control to kernel
  5. fwoskrnl.exe (Kernel)
    • Kernel initialization (kernel_init)
    • Initialize memory management
    • Initialize process scheduler
    • Initialize device drivers
    • Initialize file system
    • Start system services
  6. hal.dll (Hardware Abstraction Layer)
    • HAL initialization (hal_init)
    • Detect CPU
    • Detect memory
    • Detect devices
    • Initialize device drivers
  7. smss.exe (Session Manager)
    • Initialize user session
    • Load critical system processes
    • Start CSRSS
    • Start login manager
  8. csrss.exe (Client Server Runtime)
    • Initialize console subsystem
    • Initialize window manager
    • Start message loop
  9. freeworldlogon.exe (Login Manager)
    • Display login prompt
    • Authenticate user
    • Start user shell
  10. shell.exe (User Shell)
    • Display "FREE WORLD>" prompt
    • Accept user commands
    • Execute commands

Memory Layout During Boot

0x0000 - 0x03FF  : Interrupt Vector Table
0x0400 - 0x04FF  : BIOS Data Area
0x0500 - 0x7BFF  : Available
0x7C00 - 0x7DFF  : Boot Sector (512 bytes) - BOOTMGR
0x7E00 - 0x9FFF  : Available
0xA000 - 0xBFFF  : Video Memory
0xC000 - 0xFFFF  : BIOS ROM
0x1000:0x0000    : Kernel Entry Point

JMP Instructions

Location Instruction Purpose
BOOTMGR jmp 0x1000:0x0000 Far jump to kernel entry point
BOOTMGR (print_string) jz done Conditional jump if end of string
BOOTMGR (print_string) jmp print_string Unconditional jump to continue loop

Boot Sector Format

  • Size: 512 bytes
  • Boot Signature: 0xAA55 at bytes 510-511
  • Load Address: 0x7C00
  • Mode: 16-bit real mode
