freeworldlogon.exe - FreeWorld Login Manager

Location: services/freeworldlogon/freeworldlogon.c
Type: C Executable
Purpose: Manages user authentication and session startup

Overview

freeworldlogon.exe manages the user login process. It prompts for username and password, authenticates the user, and starts the user shell.

Main Function

int main(int argc, char** argv) {
    printf("FreeWorld Login Manager v0.1\n");
    printf("================================\n");
    
    char username[256];
    char password[256];
    
    printf("Username: ");
    fgets(username, sizeof(username), stdin);
    username[strcspn(username, "\n")] = 0; // Remove newline
    
    printf("Password: ");
    // In a real implementation, this would hide input
    fgets(password, sizeof(password), stdin);
    password[strcspn(password, "\n")] = 0;
    
    // Authenticate user (placeholder)
    printf("Authenticating...\n");
    
    // Start user shell
    printf("Starting shell...\n");
    system("shell.exe");
    
    return 0;
}

Variables

Variable Type Size Description
username char[256] 256 bytes User input buffer for username
password char[256] 256 bytes User input buffer for password

Login Process

  1. Display login banner
  2. Prompt for username
  3. Prompt for password (currently visible, should be hidden)
  4. Authenticate user (placeholder implementation)
  5. Start user shell (shell.exe)

Future Implementation

  • Password Hiding: Hide password input during typing
  • User Database: Read from /etc/passwd or similar
  • Password Hashing: Verify password against hash
  • Session Management: Create user session
  • Multiple Users: Support for multiple user accounts

Integration Points

  • smss.exe: Started by Session Manager
  • shell.exe: Starts user shell after authentication
  • /etc/passwd: User database (future)
